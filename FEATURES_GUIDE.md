# Features Quick Reference

## Where to Find Everything

### Main Page (When First Opened)
- **Upload PDF/Image** - Top section
- **Image Occlusion** - Button below upload
- **Cloze Deletion** - Button below upload
- **Bulk Import** - Button below upload

### After Generating Flashcards
You'll see tabs at the top:
- **Flashcards** - Review cards normally
- **Summary** - AI summary of material
- **Take Quiz** - Test yourself
- **AI Tutor** - Ask questions
- **Progress** - See your stats
- **Mind Map** - Visual knowledge graph (NEW!)
- **Timer** - Pomodoro study timer (NEW!)
